### Website hosted with github pages

[serekkiri.github.io](https://serekkiri.github.io/)

To run this app you need:
 - Node.js version ``8.0`` or higer

    ```
     node -v
    ```
Running up:
- You need to install depedencies:
    ```
    npm install
    ```

- Now to run execute this command:
    ```
    npm start
    ```
