# :runner: Running application

### To run this app you need:
 - Node.js version ``10.0`` or higer

    ```
     node -v
    ```
### Running up:
- First what you need to install depedencies:
    ```
    npm install
    ```

- Now to run execute this command:
    ```
    npm start
    ```
